# Personal Portfolio

A modern and responsive personal portfolio built with Next.js 16, showcasing my projects, skills, and experience.

## Features

- 🌙 Dark / Light mode
- 🌍 Multi-language support (English, Persian, Arabic)
- 📱 Fully responsive design
- ⚡ Built with Next.js 16 App Router
- 🎨 Modern UI with smooth animations
- 📂 Project showcase
- 📧 Contact section

## Tech Stack

- Next.js 16
- React
- TypeScript
- Tailwind CSS
- next-intl

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:3000
